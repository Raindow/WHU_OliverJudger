# Whu_OliverJudger_FE

## 功能模块:
### 教师
- 教师可以设置课程公告
- 支持教师自定义比赛
- 教师可以录制视频，以实现在线课程
- 教师可以选择章节题目，做为随堂检测
### 学生
- 在线答题，支持多语言C++,java,python
- 用户可以设置个人信息，查看答题成绩
